P1923 【深基9.例4】求第 k 小的数

题目描述

输入 n（1≤n<5000000 且 n 为奇数）个数字 a**i（1≤a**i<109），输出这些数字的第 k 小的数。最小的数是第 0 小。

请尽量不要使用 nth_element 来写本题，因为本题的重点在于练习分治算法。

输入格式

第一行有两个整数，分别表示 n 和 k。

第二行有 n 个整数，第 i 个数表示 ai。

输出格式

一个整数，表示第 k 小的数。

输入输出样例

输入 #1复制

输出 #1复制

说明

第二种方法是自己写的快排（写的很烂就是了，还没sort快）

快速排序

快排是一钟O(n*log(n))时间复杂度，O(n)空间复杂度的一种算法。核心思路是二分+冒泡。

具体实现

  1. 寻找一个基准点，这里是我们的p，随意找，但是我们将他作为第low个。
  2. 从头开始选，那么就从尾开始减，若j位置的数比p大，则继续遍历，j–，若比p小，则将j位置的数字填入i位置（此时i位置为p的数字，我们已经记录），再从i开始。
  3. 从i开始，若i位置的数比p小，则继续遍历，i++，若比p大，则将i位置的数字填入j位置（此时j位置的数字被填入了之前的i的位置，已经记录）再从j开始
  4. 重复3，4直到i == j之后，退出循环
  5. 执行两个新的函数，以i - 1的位置作为新的low或者high，从第一步开始，直到low >= high时返回空值。
